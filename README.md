## Installation

IpTrace is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'IpTrace'
```

## Use

```ruby
[[IpTrace shared] startWithKey:@"*******" withChannel:@"appstore"];
```
